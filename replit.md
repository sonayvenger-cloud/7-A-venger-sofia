# Hello World Python Project

## Overview
A minimal Python "Hello World" script.

## Tech Stack
- **Language:** Python 3.12
- **Platform:** Replit (Nix-based environment, `stable-25_05` channel)

## Project Structure
- `hello-world.py` — Main script that prints "Hello World!"
- `.replit` — Replit environment configuration

## Running the Project
The workflow "Start application" runs `python hello-world.py` as a console process.
